# Factorization Trainer

Welcome to the Factorization Trainer repository! This project is designed to help users learn and practice factorization skills using interactive tools hosted on Hugging Face Spaces.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/{owner}/factorization-trainer.git
   cd factorization-trainer
   ```

2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To start the application, run:

```bash
streamlit run app.py
```
